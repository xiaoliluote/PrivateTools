# PrivateTools
私人工具

使用方法：
Step 1:
Add it in your root build.gradle at the end of repositories:

     allprojects {
      repositories {
        ...
        maven { url 'https://www.jitpack.io' }
      }
    }
    
    
Step 2:
Add the dependency

    dependencies {
              implementation 'com.github.xiaoliluote:PrivateTools:0.1.1'
      }
